3D Puzzle Game

About This Game

Solve intricate 3D puzzles using physics-based mechanics. Rotate objects, manipulate gravity, and find creative solutions to challenging spatial puzzles.

Controls

  • Mouse: Rotate camera / Select objects
  • WASD: Move camera
  • Space: Interact with objects
  • R: Reset current puzzle
  • ESC: Pause menu

Features

  • Multiple challenging puzzle levels
  • Realistic physics simulation
  • Beautiful 3D graphics
  • Progressive difficulty curve
