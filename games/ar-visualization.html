AR Visualization Demo

About This Demo

Explore interactive 3D models and environments with augmented reality visualization. Manipulate, examine, and interact with detailed models in a virtual space.

Controls

  • Mouse Drag: Rotate view
  • Scroll Wheel: Zoom in/out
  • Left Click: Select object
  • Right Click: Pan camera
  • 1-5: Switch between models
  • Space: Toggle object info
  • R: Reset camera position

Features

  • High-quality 3D model rendering
  • Interactive object manipulation
  • Real-time lighting and shadows
  • Multiple visualization modes
  • Detailed model information
